# ASD4G

![ASD4G icon](Assets/app.png)

ASD4G 是一个面向游戏场景的自动化工具。它会持续监听你指定的目标程序，在程序启动时自动把显示器切换到你预设的分辨率，并在程序退出后恢复到原始分辨率。

## 功能

- Fluent 风格桌面界面，左侧提供“操作”和“设置”两个页面
- 支持添加多个目标 exe，并显示程序图标与完整路径
- 支持为每个目标程序分别设置目标分辨率
- 提供 3 组常用预设分辨率
- 支持开机自启动，适合常驻监听
- 支持简体中文、繁体中文、英文，并可继续添加 JSON 语言文件
- 当全部目标程序退出后自动还原原始分辨率

## 使用说明

1. 在“设置”页面中添加要监听的目标程序。
2. 在目标程序列表中选中对应程序，再选择或输入该程序的目标分辨率。
3. 如果你使用的是非标准分辨率，请先在 NVIDIA 或 AMD 控制面板中创建对应分辨率，并关闭缩放拉伸。
4. 回到“操作”页面启动监听，或者开启开机自启动。

## 开发

环境要求：

- .NET 10 SDK
- Windows 10/11

构建命令：

```powershell
dotnet build ASD4G.csproj -v minimal
```

运行命令：

```powershell
dotnet run --project ASD4G.csproj
```

## 语言扩展

默认语言文件位于 `Languages/` 目录。新增语言时，复制现有 JSON 文件并修改：

- `LanguageCode`
- `DisplayName`
- `Strings`

## 注意事项

- ASD4G 只能切换系统和显卡驱动已经支持的显示模式。
- 像 `2280x1280`、`1707x960`、`2027x1140` 这样的非标准分辨率，需要先由显卡驱动创建成功后才能被应用。
